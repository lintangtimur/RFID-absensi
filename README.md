# RFID-absensi
Sistem absensi mahasiswa dengan menggunakan RFID

# SystemSpec
 - PDOMysql
 - PHP v7
 - Bootstrap
# Usage[Testing]
1. Buat database dengan nama cerdas
2. Import file cerdas.sql
3. Lakukan input norfid berdasarkan table siswa di database
